## Totem

Totem is a plugin that allows you to create custom types of Totem of Undying that can only save you
from certain types of damages. You can configure totem to give custom effects, they can be crafted with custom recipes
and more!

You can read more about Totem [here](/pineapple-plugins/Totem/)
